# Online Entrepreneur Network

## Description
Online Entrepreneur Network is a console-based C++ application that enables entrepreneurs to register, log in, manage profiles, share business ideas, and connect with other entrepreneurs. The project is developed using Object-Oriented Programming (OOP) concepts and file handling for data storage.

## Features
- User Registration & Login
- Profile Management
- Share Business Ideas
- View Entrepreneur Profiles
- Connect with Entrepreneurs
- File-Based Data Storage

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
1. Compile the C++ source files.
2. Run the executable.
3. Use the menu-driven interface.
